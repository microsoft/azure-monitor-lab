# Azure Monitor Lab

Welcome to this interactive skills validation experience. Completing this module helps you prepare for the Deploy and configure Azure Monitor assessment.

In this module, you practice configuring monitoring of various workloads and infrastructure services using Azure Monitor. The lab combines both learning and hands-on practice.

## Prerequisites

To complete this lab you need an [Azure subscription](https://azure.microsoft.com/pricing/purchase-options/azure-account?cid=msft_devrel). You can apply for an Azure Free Trial subscription at: https://aka.ms/azure-free-account

## Lab Structure

This lab consists of the following exercises:

1. **[Introduction](01-introduction/index.md)** - Overview and learning objectives
2. **[Prepare Azure Environment](02-prepare-azure-environment/index.md)** - Set up required resources
3. **[Deploy Log Analytics](03-exercise-deploy-log-analytics/index.md)** - Configure Log Analytics workspace
4. **[Monitor Web Apps](04-exercise-monitor-web-apps/index.md)** - Enable Application Insights and monitoring
5. **[Configure Monitoring for Compute Services](05-exercise-configure-monitoring-compute-services/index.md)** - Set up data collection rules
6. **[Configure Alerts](06-exercise-configure-alerts/index.md)** - Create action groups and alert rules

## Estimated Time

The complete lab should take approximately **85 minutes** to complete.

## Getting Started

Begin with the [Introduction](01-introduction/index.md) to understand the skills you'll be validating in this lab.
